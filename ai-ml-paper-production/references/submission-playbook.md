# Submission Playbook

## Section Checklist

### Introduction

- problem and stakes
- exact gap
- contribution list
- result preview

### Method

- what changed from prior work
- assumptions
- enough detail to implement or reason about the method

### Experimental Setup

- datasets and splits
- baselines
- metrics
- compute or training budget if relevant

### Results

- main table or figure
- ablations tied to claims
- failure cases or limitations

### Limitations and Reproducibility

- known failure modes
- sensitivity to scale, data, or compute
- release status of code, configs, and data pointers

## Submission-Week Checklist

- Latest venue template and author instructions checked.
- All anonymization requirements satisfied.
- Claims match current figures and tables.
- Appendix and checklist are internally consistent.
- Final numbers reproduced from saved runs.
- External links follow venue policy.

## Rebuttal Template

For each reviewer concern:

1. Concern summary
2. Clarification
3. Evidence already in the paper
4. Additional analysis or text change
5. Scope adjustment if the concern is valid
